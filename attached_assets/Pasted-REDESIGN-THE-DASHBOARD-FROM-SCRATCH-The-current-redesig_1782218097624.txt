REDESIGN THE DASHBOARD FROM SCRATCH.

The current redesign has removed too much content and now feels like a generic SaaS dashboard. The platform is not just a booking system — it is a badminton ecosystem powered by AI.

The goal is to create a premium, engaging, content-rich dashboard that feels like a mix of:

* Strava Premium
* Spotify Home
* ESPN
* Formula 1
* Apple Fitness
* EA FC Ultimate Team

The dashboard should immediately make members feel connected to the badminton world when they log in.

DO NOT REMOVE:

* AI-powered Badminton News
* AI-powered Deals & Offers
* Club Announcements
* Upcoming Sessions
* Community Activity

REMOVE OR MOVE TO OTHER PAGES:

* Weather
* Location
* Polls
* Bananas Eaten
* Shuttle Counters
* Multiple Wellness Widgets
* Live Courts
* Motivational Quotes
* Any filler widgets

* Product image

* Store logo
* CTA button

£179.99

35% OFF

[ View Deal ]

VISUAL DESIGN REQUIREMENTS
==========================

The current cards feel like basic Tailwind boxes.

Completely redesign them.

Requirements:

* Premium sports platform aesthetic
* Glassmorphism
* Gradient overlays
* Layered backgrounds
* Image-rich design
* Premium typography
* Hover glow effects
* Smooth animations
* Micro-interactions
* Better spacing
* More visual hierarchy

Different card types should look different.

News cards should not look the same as deal cards.

Deal cards should not look the same as session cards.

Every section should have its own visual identity.

==================================================
LAYOUT PHILOSOPHY
=================

The issue is NOT too much content.

The issue is that all content currently has the same visual weight.

Keep the important content:

* Hero
* Deals
* News
* Sessions
* Announcements
* Community
* Performance

Make the dashboard feel like a premium badminton home screen that users want to browse every day.

When users log in, they should immediately discover:

* New deals
* Latest badminton news
* Upcoming sessions
* Club activity

The dashboard should feel like a living badminton platform, not an admin panel.
