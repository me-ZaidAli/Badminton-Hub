Build an end-to-end Referral, Acquisition Tracking, and KPI Analytics system for the Badminton Performance Group app with a premium-grade user and admin experience. Extend the signup/joining form to include a mandatory “How did you hear about us?” field with selectable options: Facebook, Instagram, TikTok, Website, Word of Mouth, Leisure Centre, Saw a Session Running, Through a Coach, Referral Link/Code, and Other (with free text); this value must be stored at account creation and locked from user edits. If a user joins via a referral link or referral code, the system must automatically set the acquisition source as “Referral” while still retaining internal attribution data. Implement a Referral System where users can generate unique, single-use referral codes and shareable referral links from a dedicated, visually impressive Referrals dashboard (accessible via main menu, profile, and optional dashboard promo card); each code expires after 30 days, shows a live countdown, and has a status (Active, Used, Expired). Referral links must redirect to the signup/login page, which includes an optional referral code field; the system must validate that the code exists, is unused, not expired, and matches the referred user’s name and email before allowing referral-based registration. Upon successful signup, log the referral as Pending Approval and automatically send the referrer an internal chat message thanking them, plus an in-app notification confirming referral receipt. Admins must manage referrals from a dedicated Admin Referrals tab, where approving a referral marks the code as used, adds £4 credit to the referrer’s credit wallet, logs the action, updates referral progress, and triggers both an internal chat message and notification confirming approval and credit awarded. Credits must accumulate automatically, with system-tracked thresholds: £8 credit unlocks eligibility for a 4-month Premium member rate and £16 reflects four approved referrals; users and admins must be notified and flagged when thresholds are reached, with admins applying benefits manually. Create an Admin-only Acquisition & KPI Analytics panel with modern charts and tables showing signups per month, signups by acquisition channel, month-on-month growth, Premium conversion rate by channel (Premium users from channel ÷ total users from channel), average time to Premium by channel, retention rate by channel (users active after 90 days ÷ users joined via channel), average active lifespan per channel, referral effectiveness, organic acquisition ratio, and a weighted Channel Quality Score combining Premium conversion, retention, and activity. Define “active user” as one who has logged in, signed up for a session, or submitted match data within the last 30 days. Add filters by date range, club/location, membership type, and acquisition source, plus CSV export. Automatically generate a Monthly Admin Summary Report on the 1st of each month showing growth overview, acquisition breakdown, Premium insights, retention insights, referral performance, and system-generated recommendations (e.g. best-performing channel, fastest-growing channel, highest-retention source), visible in the admin dashboard and optionally emailed. Ensure duplicate referrals are prevented via email checks, all actions are audit-logged, permissions are enforced, expired referral codes trigger user notifications, and the entire system feels polished, transparent, data-driven, and premium for both users and admins.
