3 RIVALRY ANALYTICS (DERIVED FROM MATCH SCORES)

Using match score data, calculate the following advanced metrics.

Total Points Scored

Sum all points from every match.

Example:

Haider: 52 points
Tarik: 63 points

Display as horizontal comparison bars.

Average Points Per Match

Formula:

Total points / number of matches

Example:

Haider: 17.3
Tarik: 21.0

Display with small stat cards.

Average Score Margin

Calculate the average difference between scores in each match.

Formula:

|score difference|

Example:

21–13 → margin 8
21–19 → margin 2

Average margin = 4.3

Display this metric with a rivalry intensity label:
