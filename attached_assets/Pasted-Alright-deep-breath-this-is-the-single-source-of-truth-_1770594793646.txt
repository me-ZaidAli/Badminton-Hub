1. GLOBAL ROLE DEFINITIONS

System Roles (Global)

Super Admin

Admin

Organiser

Coach

Player


⚠️ Important principle:

> Admins, organisers, and coaches are also players by default.



Roles are additive, not exclusive.


---

2. CLUB ROLE MODEL (CRITICAL)

Roles are club-specific, not global.

A single user can be:

Owner of Club A

Admin of Club B

Player of Club C


These roles must never override each other.


---

3. CLUBS SYSTEM RULES

3.1 Club Ownership

The user who creates a club becomes:

Club Owner


Owner has full control over:

Admin promotion

Member approvals

Club settings




---

3.2 Club Admins Are Also Players

Club admins:

Appear as players in sessions

Appear in rankings

Can join other clubs as players


Admin privileges are in addition to player abilities



---

3.3 Joining Clubs

Own Club

If user owns the club:

❌ No “Join” button

❌ No join request

Already a member by default


Other Clubs

Users (including admins) can:

Submit join request

Request is reviewed by:

Club Owner / Admins



On acceptance:

User is added as Player only

❌ No automatic admin rights


Admin promotion:

Only by club owner

Manual action only



---

3.4 Multiple Clubs

Unlimited club memberships allowed

Roles are isolated per club

Leaving one club does not affect others
